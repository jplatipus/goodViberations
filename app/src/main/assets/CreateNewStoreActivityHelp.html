This page is used to create a new password store file. The file is encrypted, using the password and memorable date as keys to encrypt the file.

The password should be something that is easy to remember by you. Upper case and lower case characters are different, so care needs to be taken if text case is present in the password. The password should have a minimum of eight characters.

The memorable date field should be a date that you find easy to remember, a birthday or something similar are usually good candidates.

The password and memorable date values are entered twice to help ensure that the values entered are those wanted, and that they do not have an error that has not been noticed.

help displays the help for the page being displayed, in this case this help is displayed, which explains the steps necessary to create a new encrypted password store that can hold your passwords.

The back button returns to the previous page (initial options)

The create new store button creates a new password store, and displays the next page, which is the list of entries in the store, and the ability to add new passwords to the list.
