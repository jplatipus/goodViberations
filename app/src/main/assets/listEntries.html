The list entries page lists all the different entries that have been added: Internet login, credit card details...

The list displays the entry name of each entry.

The following actions can be performed on this page:

  • When an entry is tapped, a new page with all the entry details, is displayed.
  • New entries can be added by tapping the (+) in the toolbar at the bottom of the page.

Entry Types

When adding a new entry, this app prompts for the kind of entry to create. Depending on the type selected, an initial set of fields are prepared. For example, a credit card entry contains a PIN value, whereas a User Password entry does not.

The following types are available:

  • User, Password is the most common type.
  • User, Password, Id is useful for websites which have a combination of username, id and password.
  • Credit Card is aimed at storing credit card information.
  • Encrypt Me rarely used, just here for completeness.
The following set of fields is common to all entry types:
  • Name, the name that appears in the list of entries
  • Notes, this field is there to allow free text to be added to the entry
  • Web Url, is used to associate the entry with a web address.
