The options presented are used as follows:

Open Store is used to open an existing password store file. If there is no file present, this usually happens when first using the app, the button is disabled.

Create New Store is used to create a new password store file. If there is a file present, the button is disabled.

clear existing store is used to delete the existing password store file. If there is no file present, this usually happens when first using the app, the button is disabled.

import/export store imports or exports a password store that is in an unencrypted xml format.

about displays information about this app, such as the copyright, version number.

help displays the help for the page being displayed, in this case this help is displayed, which explains the options that are available when the app is launched and the first page is displayed.
